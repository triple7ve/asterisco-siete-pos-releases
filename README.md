# Asterisco Siete

### POS release distribution

Repository dedicated exclusively to **storing and distributing the versions** of the **Asterisco Siete** point-of-sale application.

</div>

---

## 📦 What is this repository?

This repo **contains no source code**. Its only purpose is to act as the **official distribution channel** for the installers and update manifests of the *Asterisco Siete* POS application.

Every published version is uploaded as a **GitHub Release**, and from there:

- 🧑‍💼 Users download and install the application.
- 🔄 The installed app checks this repo to **detect and apply updates automatically**.

> [!NOTE]
> If you're looking for source code, business logic, or development issues, this is **not** the place. Only ready-to-install binaries live here.

---

## 🚀 Installation

1. Go to the **[Releases](https://github.com/triple7ve/asterisco-siete-pos-releases/releases)** section.
2. Download the latest installer: `asterisco-siete-<version>-setup.exe`.
3. Run the installer and follow the steps.

Once installed, **you don't need to download manually again**: the application keeps itself up to date. ✨
